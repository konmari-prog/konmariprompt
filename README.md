# Nano Banana Pro クリスマスプロンプト集前編

クリスマスをテーマにした画像変換プロンプト集

## 概要

このリポジトリには、クリスマスをテーマにした6つの画像変換プロンプトが含まれています。各プロンプトは、入力画像を様々なクリスマス風の製品写真に変換します。

## プロンプトカテゴリ

1. **アクリルLEDスタンド編** - Image-to-Acrylic LED stand
2. **クリスタル球体LED編** - Image-to-Crystal ball LED light
3. **特注ケーキ編** - Image-to-Cake
4. **アクセサリー編** - Image-to-Jewelry
5. **スノードーム編** - Image-to-Snow Globe
6. **ジンジャーブレッドハウス編** - Image-to-Cookie House

## 使い方

1. `index.html`をブラウザで開く
2. 各カテゴリのプロンプトを確認
3. 「プロンプトをコピー」ボタンでプロンプトをコピー
4. お好みのAI画像生成ツールで使用

## ファイル構成

```
konmari/
├── index.html          # メインページ
├── prompts.json        # プロンプトデータ
└── images/             # 各カテゴリの代表画像
    ├── acrylic_led_stand.jpg
    ├── crystal_ball_led.jpg
    ├── cake.jpg
    ├── jewelry.jpg
    ├── snow_globe.jpg
    └── cookie_house.jpg
```
