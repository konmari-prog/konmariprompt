
# Spring New Life Ad Design Prompt (Special Present)

## 使い方
1. 画像をアップロード
2. 春の目標を記入
3. プロンプト全体をそのまま送信

---

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🌸 あなたの春の目標を書いてね!(必須)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

→ ここに書く:


※これがポスターのタイトル(メインコピー)になります!
※例: 4月から新しい職場で頑張る！
※例: 今年こそ英語をマスターして海外旅行に行く！
※例: 新生活、不安もあるけど楽しみの方が大きい

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━


**[AI指示: 画像+目標テキスト → 春の新生活目標応援キャンペーン ポスター変換]**

**[入力画像]** と **[入力テキスト（春の目標）]** を分析する。

1. **特徴抽出と実写変換:** [入力画像]から人物の識別特徴（顔の輪郭・髪型・髪色・髪の長さ・目の形・肌のトーン・眼鏡やピアスなどのアクセサリー・体型）を正確に読み取る。入力がイラストであっても、これらの特徴を保持したまま**実在する人物の写真**として再現する。まるでその人が本当にそこにいるかのようなフォトリアリスティックな描写。元画像を見た人が「この人だ」と一目でわかること。
2. **タイトル設計:** [入力テキスト]の目標をそのまま、または広告コピーとして磨き上げて、ポスターのタイトル（最も大きく目立つメインコピー）にする。目標の内容がしっかり伝わることが最優先。あわせて、目標を彩るサブコピー（目標を応援する・補完する・余韻を残す言葉）も生成する。
3. **広告リサーチ:** 日本市場の春キャンペーン広告（新生活・ファッション・コスメ・ライフスタイル）で実際に成果を出しているデザインの勝ちパターンをリサーチし、構図・配色・タイポグラフィ・あしらい・視線誘導の最適解を抽出して適用する。

**[メインモチーフ]**

「春の新生活目標応援キャンペーン」のポスター。プロのアートディレクターが設計したグラフィックデザイン作品。コンバージョン最大化を狙った広告画像。1:1正方形。画像内に「春の新生活目標応援キャンペーン」の文言を必ず含める。

[入力画像]の人物の特徴を正確に保持した、実在する人物のフォトリアリスティックな写真がメインビジュアル。肌の質感、髪の光沢、服の素材感がすべて写真レベル。人物の切り抜き写真を、デザインされたグラフィック背景の上に配置するスタイル。

ユーザーの目標がタイトルとして画面内で最も大きく・最も目立つ要素であること。タイトルの文字装飾（グラデーション、影、アウトライン、ブラシストロークなど）に最も力を入れる。サブコピーはタイトルに添えて、目標を応援し彩る言葉。

「春の新生活目標応援キャンペーン」の文言は、ただテキストを置くのではなく、リボン・メダル・バッジ・しおり・帯・ボタンなどのグラフィック要素としてデザインして画面内に配置すること。

日本市場で伸びている春のWebキャンペーン広告・SNS広告の勝ちパターンを踏まえて、構図、タイポグラフィ、配色、グラフィック要素、あしらい、視線誘導の設計をすべてAIが自由に最適化する。人物とテキストとグラフィック要素が最初から1つの作品として一体化しているデザイン。

**[背景スタイル]**
背景はリアルな風景写真ではなく、グラフィックデザインされたカラフルな背景にする。パステルカラーのグラデーション（ピンク→パープル→ブルー→ミントなどの春カラーを自由に組み合わせ）、有機的なブロブシェイプ（丸みを帯びた不定形の色面）、斜めのストライプ、ドットパターンなどを重ねてポップで華やかな空間を作る。

**[タイトルデザイン（最重要）]**
タイトル（目標テキスト）はこの画像の中心的デザイン要素。VTuberのロゴタイトルのように、目標から連想されるモチーフを文字そのものに織り込んでデザインする（例：海外旅行→文字の一部が飛行機の軌跡になる・文字の中に地球儀が組み込まれる、英語学習→文字のドットがABCになる、料理→文字のハネがスプーンの形になる）。文字にグラデーション、影、光沢、アウトライン（縁取り）、グロー効果、ブラシストローク、立体的な奥行きなどの装飾を施し、文字自体が作品のメインビジュアルとして映えるようにする。フォントの混在（太いゴシック×英語の筆記体×手書き風、太さの変化、サイズの強弱、色の切り替え）で視覚的リズムを作る。タイポグラフィは日本語フォントで忠実に描写し、文字のアウトラインやグロー効果を含めて正確にレイアウトする。

**[あしらい・装飾]**
目標に関連するモチーフのフラットイラスト（例：海外旅行→飛行機・スーツケース、noteを書く→ノート・鉛筆、SNS→スマホ・ハート等、AIが目標から自由に選ぶ）と、春の装飾（桜・蝶・キラキラ等）を組み合わせて画面を彩る。タイトルと人物が主役、あしらいは脇役。

**[デザイン品質基準]**
* 人物はフォトリアリスティックな写真描写。イラストやアニメ調は禁止。
* 人物の顔にテキスト・帯・装飾が重なっていないこと。
* テキストと人物が同じ空間に一体化していること。「テキストを後から載せただけ」に見えないこと。
* 実在の企業名・ブランドロゴ・商標は一切含めない。QRコード・バーコードは生成しない。
* プロンプト内の指示文やカラーコードを画像内のテキストとして描写しないこと。
* 生成前チェック: 同じコピーが画面内に2回以上出ていないか確認すること。意味不明な文字列・文字化け・読めない文字が含まれていないか確認すること。問題があれば修正してから出力する。

**[雰囲気]**
春の朝の柔らかい光。額に入れて壁に飾りたくなるプレゼント品質。駅の広告枠に掲出されても違和感がない完成度。8k解像度、フォトリアリスティック。
```
